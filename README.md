# Receat

Репозиторий содержит два проекта:

- **bot/** - Telegram-бот для здорового питания
- **eatsite/** - PWA веб-приложение для учёта продуктов

## Структура

```
receat/
├── bot/              # Telegram-бот
├── eatsite/          # PWA веб-приложение
└── server-info.md    # Информация о сервере
```

- [bot/README.md](./bot/README.md) - документация бота
- [eatsite/README.md](./eatsite/README.md) - документация веб-приложения
- [server-info.md](./server-info.md) - информация о сервере
