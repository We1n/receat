# Инициализация продуктов

## Быстрый старт

Для инициализации workspace с базовым набором продуктов выполните:

```bash
npm run init-products <workspace_id>
```

Например:
```bash
npm run init-products test-pantry
```

Или напрямую:
```bash
node scripts/init_products.js test-pantry
```

## Что делает скрипт

1. Создает или обновляет workspace с указанным ID
2. Добавляет ~60 базовых продуктов по категориям:
   - Овощи (10 продуктов)
   - Фрукты (6 продуктов)
   - Молочные продукты (8 продуктов)
   - Мясо (6 продуктов)
   - Крупы (9 продуктов)
   - Специи (8 продуктов)
   - Напитки (6 продуктов)
   - Прочее (8 продуктов)

3. Все продукты изначально помечены как `in_stock: false` (нужно купить)

## Использование

После инициализации:
1. Откройте сайт: `http://localhost:5173`
2. Введите `workspace_id`, который использовали при инициализации
3. Нажмите "Присоединиться"
4. Вы увидите список продуктов, которые нужно купить
